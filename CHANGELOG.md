# CHANGELOG

---

> Здесь собраны все изменения проекта по версиям. Формат основан на принципах **Keep a Changelog**.

---

## v0.1.0 — Первый релиз (2025-11-19)

### Добавлено

- Базовая структура приложения
- Поддержка создания новой заметки
- Поле для редактирования названия заметки
- Навигация между страницами через React Router
- Хранение заметок через `pagesContext`
- TailwindCSS v4 для стилей

---

## v0.1.1 — Эмодзи пикер (2025-11-20)

### Добавлено

- Компонент `EmojiPicker`
- Grid-сетка эмодзи
- `Frequently Used` список
- Поиск по эмодзи (UI готов, логика в процессе)
- Открытие и закрытие через позиционирование по кнопке

### Изменено

- Кнопка "Add Icon" теперь заменяется выбранным эмодзи
- Улучшено позиционирование попапа (учёт scrollX/scrollY)

---

## v0.2.0 — Добавления обложки заметки (2025-11-20)

### Добавлено

- Кнопка "Add Cover" с загрузкой изображения
- Автопозиционирование обложки над редактором
- Поддержка `object-cover + object-top` для отображения верхней части изображения

### Изменено

- Улучшен UI кнопок в шапке редактора

### Фикс багов

- `A component is changing an uncontrolled input to be controlled` предупреждение
- Неправильная высота изображения при использовании `h-100`

---

## v0.3.0 — Изменение UX (2025-11-21)

### Добавлено

- Тени, скругления и визуальная иерархия элементов
- Более плавный UX у EmojiPicker

### Изменено

- Оптимизация ререндера EmojiPicker через React.memo

## v0.4.0 — Улучшение боковой панели (2025-11-22)

### Добавлено

## v0.5.0 — Улучшение работы с контентом (2025-11-24)

### Добавлено

- Компонент `BlockContent.jsx` для рендера блока контента в заметке
- Компонент `CommandTips.jsx` для подсказки команд в поле добавления

### Изменено

- Изменён дизайн поле добавления добавлена кнопка для добавления через клик

## v0.6.0 — Улучшение системы контент-блоков (2025-11-26)

### Добавлено

- Компонент `BlockList.jsx` для всех видов списков
- Компонент `HeadingH1.jsx` для заголовка 1 уровня
- Компонент `HeadingH2.jsx` для заголовка 2 уровня
- Компонент `HeadingH3.jsx` для заголовка 3 уровня
- Компонент `Underline.jsx` для текста с нижним подчёркиванием
- Компонент `Paragraph.jsx` для обычного текста

### Улучшено

- Полностью реализованы команды для создания разных блоков с контентом
